# Souvra

Souvra is a platform-neutral suite of deterministic tools and portable agent skills for working with local Open Knowledge Format (OKF) bundles.

This repository is the public home of the Souvra CLI, portable skills, shared contracts, fixtures, documentation, and release artifacts.

## Status

Planning and design are in progress. Implementation has not started yet.
